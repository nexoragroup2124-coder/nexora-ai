## Text Content Needed

### Privacy Policy
Create file: `privacy.html` or link to external service

Example placeholder:
```html
<h1>Privacy Policy</h1>
<p>Nexora AI respects your privacy. We do not collect personal data beyond what is necessary to provide the service.</p>
<h2>Data Collection</h2>
<p>- We do not store chat conversations on our servers by default</p>
<p>- Your data stays on your device</p>
<p>- Optional cloud backup (when enabled by user)</p>
```

### Support Page

<h1>Support</h1>

## Testing Checklist Before Submission

- [ ] App runs on iOS 13+ and Android 8+
- [ ] App works in all languages
- [ ] No crashes on startup
- [ ] Chat feature works offline (PWA)
- [ ] All buttons functional
- [ ] No external links are broken
- [ ] Privacy policy accessible
- [ ] Support email/form works
- [ ] App installs from home screen
- [ ] Icons display properly
- [ ] Text is readable
- [ ] Performance is smooth (no lag)

---

## Common Rejection Reasons & Fixes

| Reason | Fix |
|--------|-----|
| Crash on launch | Test thoroughly, check logs |
| Broken links | Verify all external URLs work |
| Missing privacy policy | Add privacy.html to app |
| Unclear purpose | Improve app description |
| Poor UI/UX | Ensure high-quality screenshots |
| Misleading promises | Review description accuracy |
| Requires account | Allow usage without signup |
