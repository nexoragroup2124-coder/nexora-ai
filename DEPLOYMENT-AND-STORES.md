# Nexora AI - Deployment & App Store Setup Guide

This guide walks you through deploying Nexora AI to the web and making it available on Play Store and App Store.

---

## Phase 1: Deploy to Netlify (5 minutes)

### Step 1: Push to GitHub

1. Open GitHub: https://github.com
2. Sign in (create account if needed)
3. Click **New repository**
4. Name it: `nexora-ai`
5. Choose **Public**
6. Click **Create repository**
7. Clone it locally:
   ```bash
   git clone https://github.com/YOUR-USERNAME/nexora-ai.git
   cd nexora-ai
   ```
8. Copy all Nexora files into this folder
9. Commit and push:
   ```bash
   git add .
   git commit -m "Initial Nexora AI deployment"
   git push origin main
   ```

### Step 2: Deploy via Netlify

1. Go to: https://app.netlify.com
2. Sign in or create account (use GitHub login for easier setup)
3. Click **Add new site** → **Import an existing project**
4. Select **GitHub**
5. Authorize & choose your `nexora-ai` repository
6. **Build command:** Leave empty
7. **Publish directory:** `.` (root folder)
8. Click **Deploy site**
9. Wait 1-2 minutes for deployment
10. Netlify assigns you a URL like: `https://random-name-12345.netlify.app`

**Netlify automatically rebuilds when you push updates to GitHub.**

## Phase 2: Custom Domain Setup (Aesthetic URL)

### Option A: Free Subdomain (Recommended for Testing)

1. In Netlify dashboard, go to **Site settings** → **Domain management**
2. Click **Add custom domain**
3. Enter: `nexora-ai.netlify.app`
4. Netlify auto-assigns this subdomain

**Your public URL:** `https://nexora-ai.netlify.app`

### Option B: Paid Custom Domain (nexora-ai.app)

1. Buy domain from: **Namecheap**, **GoDaddy**, or **Google Domains**
   - Search for: `nexora-ai.app` or `nexora-ai.io`
   - Cost: ~$10-15/year
   
2. In Netlify:
   - Go to **Site settings** → **Domain management**
   - Click **Add custom domain**
   - Enter your domain (e.g., `nexora-ai.app`)
   - Netlify shows DNS records to add to your registrar

3. At your domain registrar:
   - Add the Netlify DNS records
   - Wait 24-48 hours for DNS propagation

4. Enable free SSL:
   - Netlify auto-enables HTTPS
   - Verify in **Site settings** → **SSL/TLS certificate**

**Result:** Your app is now at `https://nexora-ai.app`

## Phase 4: Installable App - Google Play Store

### Requirements
- Google account
- $25 one-time registration fee
- App signing certificate

### Step 1: Build Android App Bundle

1. Install required tools:
   ```bash
   npm install -g @capacitor/cli
   npm install -g cordova-res
   ```

2. Create Capacitor project:
   ```bash
   cd /path/to/nexora-ai
   npx cap init nexora-ai com.nexora.ai
   ```

3. Add Android:
   ```bash
   npx cap add android
   ```

4. Update `capacitor.config.json`:
   ```json
   {
     "appId": "com.nexora.ai",
     "appName": "Nexora AI",
     "webDir": "."
   }
   ```

5. Build for Android:
   ```bash
   npx cap build android
   ```

6. Open Android Studio:
   - File → Open → `android` folder
   - Build → Build Bundle(s) / APK(s) → Build Bundle(s)
   - Generate signed bundle with your keystore

### Step 2: Submit to Play Store

1. Go to: https://play.google.com/console
2. Create new app
3. Fill in:
   - App name: `Nexora AI`
   - Category: **Productivity**
   - Content rating: **4+**
   - Description: "Free AI chat assistant for research, writing, coding, and planning"
   - Screenshots (3-5 images of the app)
   - Feature graphic (1024x500px)
   - Icon (512x512px)

4. Upload signed bundle
5. Review pricing: **Free**
6. Submit for review

**Timeline:** 1-7 days for approval

## Phase 5: App Store (iOS)

### Requirements
- Apple Developer Account ($99/year)
- Mac with Xcode
- App signing certificate

### Step 1: Build iOS App

1. Install CocoaPods (if needed):
   ```bash
   sudo gem install cocoapods
   ```

2. Add iOS to Capacitor:
   ```bash
   npx cap add ios
   ```

3. Build for iOS:
   ```bash
   npx cap build ios
   ```

4. Open Xcode:
   ```bash
   open ios/App/App.xcworkspace
   ```

5. In Xcode:
   - Select project → Signing & Capabilities
   - Choose your Apple Team
   - Update bundle ID: `com.nexora.ai`
   - Build archive: Product → Archive

### Step 2: Submit to App Store

1. Go to: https://appstoreconnect.apple.com
2. Create new app
3. Fill in:
   - App name: `Nexora AI`
   - Bundle ID: `com.nexora.ai`
   - SKU: `nexora-ai`
   - Category: **Productivity**

4. Add app information:
   - Description: "Free AI chat assistant for research, writing, coding, and planning"
   - Keywords: `nexora, ai, chat, assistant, chatbot`
   - Support URL: `https://nexora-ai.app`
   - Screenshots (5 images, 2 required for each device)
   - App preview video (optional, 15-30 sec)

5. Set pricing: **Free**
6. Upload build
7. Submit for review

**Timeline:** 1-3 days for approval

## Troubleshooting

### App not appearing in search?
- Wait 2-7 days for Google indexing
- Check Google Search Console for errors
- Submit sitemap manually

### Build fails on Android/iOS?
- Ensure Node.js v16+ installed
- Clear node_modules: `rm -rf node_modules && npm install`
- Check platform-specific prerequisites

### Custom domain not working?
- Verify DNS records propagated (24-48 hours)
- Clear browser cache
- Test with: `https://dns.google`

### App rejected from stores?
- Review rejection reason
- Common issues: app store guidelines, content policy
- Resubmit after fixes
