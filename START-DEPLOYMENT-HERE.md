## ❓ FAQ

**Q: How much does this cost?**
A: 
- Netlify deployment: FREE
- Custom domain: ~$10-15/year (optional)
- Play Store: $25 one-time
- App Store: $99/year
- Total minimum: FREE (just use Netlify domain)

**Q: How long until I see it on Google?**
A: 2-7 days typically. Check Google Search Console to verify indexing status.

**Q: Can users search "nexora ai" and find my app?**
A: Yes! On:
- Google Search (in 2-7 days)
- Yahoo Search (in 2-7 days)
- Play Store (after 1-7 days approval)
- App Store (after 1-3 days approval)

**Q: What if I don't have a Mac for iOS?**
A: You can still submit Android to Play Store. For iOS, you need a Mac with Xcode. Many developers use cloud services like Mac Rite or GitHub Actions to build iOS apps.

**Q: Can I keep deploying updates?**
A: Yes! Push to GitHub → Netlify auto-deploys in seconds. App store updates take 1-7 days approval.

**Q: Is the app really free for users?**
A: Yes! Your frontend is free. Users provide their own API key or you add a backend (optional).
