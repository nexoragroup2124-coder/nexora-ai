# 🚀 DEPLOY NEXORA AI IN 5 MINUTES

Follow these steps to get your app live at `nexora-ai.app` with Google/Yahoo searchability.

---

## STEP 1: Push to GitHub (2 min)

```bash
# In your nexora-ai folder:
git init
git add .
git commit -m "Initial Nexora AI"

# Go to https://github.com/new
# Create repo: nexora-ai (PUBLIC)
# Then:
git remote add origin https://github.com/YOUR-USERNAME/nexora-ai.git
git branch -M main
git push -u origin main
```

---

## STEP 2: Deploy on Netlify (2 min)

1. Go to **https://app.netlify.com**
2. Click **Sign up** → Choose **GitHub**
3. Click **Add new site** → **Import from Git**
4. Select your `nexora-ai` repo
5. Settings:
   - **Build command:** (leave empty)
   - **Publish directory:** `.`
6. Click **Deploy site**
7. Wait ~2 min... You'll get a Netlify URL! 🎉

**Example:** `https://your-nexora-12345.netlify.app`

---

## STEP 3: Add Custom Domain (1 min)

### Free option (Fastest):
1. In Netlify dashboard → **Site settings** → **Domain management**
2. Click **Add custom domain**
3. Type: `nexora-ai.netlify.app`
4. Done! ✅

**Your new URL:** `https://nexora-ai.netlify.app`

### Premium option (Better branding):
1. Buy domain: `nexora-ai.app` (~$10/year)
   - Use: https://namecheap.com or Google Domains
2. In Netlify, add this domain
3. Update DNS at registrar
4. Wait 24 hours for DNS sync

**Your new URL:** `https://nexora-ai.app`
